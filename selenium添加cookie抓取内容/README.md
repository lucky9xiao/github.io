## 关于网站爬虫识别



**网站一般性用户识别分级**

- ​	正常用户：可直接访问网站
- ​	可疑用户：识别给出的验证码后可访问网站
- ​	恶意用户：识别给出的网站后，同样不能访问网站



## 关于selenium.webdriver设置

**开发者模式：**使用此模式可以较大程度上防止被网站识别。以tb网为例，若不开启开发者模式，直接被判定为恶意用户。
